Conditional Statements
~~~~~~~~~~~~~~~~~~~~~
[Old Version]  --  *ngIf Directive 
	<div *ngIf="isLoggedIn">Welcome back!</div>

[New  Version]  ---   @if block

	@if (isLoggedIn) {
			<div>Welcome back!</div>
	}
	
	
	//  If..Else
	@if (isLoggedIn)
   {
			<div>Welcome back!</div>
	} 
	@else 
	{
		  <div>Please log in.</div>
	}

	


Iterate Collections in Template
~~~~~~~~~~~~~~~~~~~~~~~~~~~

[Old Version]

	<tr  *ngFor="let item of empsArray">
				<td>{{item.empno}}</td>
			   .....
    </tr>


[Latest Version]   ---  @for  block

 @for (item of empsArray; track item.empno) 
    {
       <tr>
				<td>{{item.empno}}</td>
			   .....
    </tr>
    }


Eg2:
<ul>
    @for (item of skillsArray; track $index) {
        <li> {{item}} </li>
    }   
</ul>




Note:  @for block is Introduced in Angular 17, it replaces the traditional *ngFor directive with a more performant and concise syntax
